常見陷阱：數字 '0' 的值

常見陷阱：偶同位定義

偶同位是讓「1 的總數」為偶數，不是讓檢查碼固定為 0。

標準 ASCII
口訣：美國標準=7bits
 7 bits 通常佔 1 Byte
(第8位元閒置)		 2^7 = 128 美國英語
! 陷阱:常誤以為標準 ASCII 用滿 8 bits (256字)，其實只有 7 bits (128字)。
Unicode
口訣：萬國碼=2Bytes
 16 bits (基礎) 2 Bytes 2^16 = 65536 全球多國語言
! 陷阱:考試若無特別說明，Unicode 通常指 UCS-2 (2 Bytes) 而非 UTF-8。
Big-5
口訣：台灣舊標準
 16 bits 2 Bytes 約 1.3 萬字 繁體中文 (舊)
! 陷阱:雖與 Unicode 同為 2 Bytes，但編碼不同，同一字碼值不同。
